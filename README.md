# assignment-01：用 Cursor 创建你的个人介绍页面 🎨
## 👋 欢迎

这是你的第一个作业！你将使用 Cursor 的 AI 辅助功能，创建一个属于你自己的个人介绍页面。

**预计用时**：1-2 小时  
**难度**：⭐⭐☆☆☆

---

## 🎯 任务要求

### 必须包含（60分）

- [ ] **个人信息**：你的名字和一句话介绍
- [ ] **头像照片**：一张你的照片或卡通头像
- [ ] **技能展示**：至少 3 个你的技能或兴趣爱好
- [ ] **联系方式**：邮箱或社交媒体链接（至少1个）

### 加分项（40分）

- [ ] 页面配色协调，布局美观（15分）
- [ ] 有动画效果，如鼠标悬停、渐入效果（10分）
- [ ] 响应式设计，手机上显示正常（10分）
- [ ] 有独特的创意元素（5分）

---

## 🚀 开始之前

### 需要准备的工具

1. **Cursor 编辑器**
   - 下载地址：https://cursor.sh
   - 用你的 GitHub 账号登录
   - 免费版即可完成作业

2. **Git**（用于提交作业）
   - Windows: https://git-scm.com/download/win
   - Mac: 打开终端输入 `git --version`（会自动提示安装）

3. **浏览器**（Chrome/Edge/Safari 都可以）
